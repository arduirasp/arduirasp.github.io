Installation de l'IDE Arduino

Published: mer. 30 décembre 2015
By Dominique

In Arduino.

tags: installation

Informations

L'IDE (logiciel pour le code) ainsi que les schémas de la carte sont sous licence libre. Le nom Arduino a été déposé et a entraîné un réglement au tribunal. Nous n'allons pas nous étendre mais il existe de ce fait maintenant deux branches : Arduino.cc et Arduino.org . De ce fait, il existe plusieurs cartes mais aussi logiciels pour le code. Nous nous concentrerons uniquement sur la branche Arduino.cc qui est la branche originelle dont le nom en France est Genuino (au lieu d'Arduino pour ces problèmes de nom déposé).

La carte phare

Il s'agit du modèle Uno, modèle idéal pour commencer à bidouiller.

Que faire ?

Réponse courte, presque tout. Détecter, commander des moteurs, allumer des DEL...

Installation

Pour pouvoir télécharger l'IDE, rendez vous sur Arduino.cc. Télécharger la version correspondante à votre distribution.

Windows

Pour plus de facilité, télécharger la version \emph{installer}. Cette version contient les drivers nécessaires au dialogue entre la carte Arduino et le PC. A la fin de l'installation, vous aurez l'entrée dans le menu ainsi qu'un raccourci sur le bureau.

Mac Os X

Télécharger la version correspondante. Une fois décompressée, l'archive libère l'application Arduino que vous pouvez installer où vous voulez.

GNU/Linux

La plupart du temps, les distributions possèdent un dépôt de logiciels. Bien qu'Arduino soit présent, c'est une version ancienne est disponible. Pour obtenir la dernière version, télécharger la version 32 ou 64 bits correspondant à votre système. Avant de décompresser l'archive, l'IDE Arduino dépend de plusieurs paquets : gcc-avr, avr-libc, openjdk-8jdk. La décompression de l'archive fait apparaître un dossier nommé arduino-1.6.6 (selon la version de l'IDE). L'utilisateur doit être dans le groupe tty et dialout. Pour rendre ce changement effectif, il suffit de se reconnecter. Dans un terminal, il suffit de se placer dans ce dossier et de lancer ./arduino

Si d'aventure, l'installation n'est pas possible, le mieux est de consulter les sites relevant de votre distribution ou de chercher sur un moteur de recherche l'erreur.
