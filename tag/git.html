B-A BA de git

Published: dim. 14 février 2016
By Dominique

In GNU/Linux.

tags: git

Qu'est-ce ?

Git est un système de versionning. Il s'agit donc de garder en mémoire toutes les versions d'un fichier avec donc toutes les modifications. Pour plus d'informations, voir la page git sur wikipédia.

Avec ce système, il est facile et simplement de revenir sur une modification ; par exemple, dans un texte si on change un paragraphe mais qu'ensuite, la modification n'est pas correcte, le retour en arrière est facilité.

Nous allons voir dans ces lignes comment utiliser un dépôt sur notre ordinateur en utilisant la ligne de commande, plus facile et rapidement.

Installation

installer git en utilisant le gestionnaire de paquets, par exemple sous un système deb :

sudo apt-get install git

Paramétrer nom et mail avec la commande 

git config --global user.name "nom"
git config --global user.email mail@mail.com

Constuire le dépôt

Créer le dépôt dans le répertoire choisi (il ne doit pas être nécessairement vide) :

git init

Le premier commit

Un fichier readme.md est créé. Nous allons donc créer un premier commit en ajoutant le fichier dans le suivi git et en le prenant en compte dans le suivi avec une description

Ajouter le fichier

git add readme.md
ou
git add *

Prendre en compte le changement (ajout du fichier) dans le versionning

git commit -m "nom du commit"
