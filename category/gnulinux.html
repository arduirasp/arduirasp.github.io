Exclure des fichiers/dossiers du suivi Git

Published: lun. 15 février 2016
By Dominique

In GNU/Linux.

tags: git

A quoi sert ce fichier ?

Gitignore est un fichier caché avec le point devant qui définit les règles de suivi du dépôt Git. Lorsque des fichiers ou dossiers notamment temporaires ne sont pas nécessaires dans le suivi, ils sont spécifiés dans ce fichier gitignore.

Ce fichier est à placer à la racine du dépôt.

Exclure/inclure un fichier/dossier dans le suivi

Ignorer tous les fichiers nommés todo.txt

Ecrire dans le ficher .gitignore :
todo.txt

Ignorer tous les fichiers txt

Ecrire dans le ficher .gitignore :
*.txt

Réinclure un fichier à la suite de l'exclusion d'ensemble précédente

Ecrire dans le ficher .gitignore :
!foo.html
